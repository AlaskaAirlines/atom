public extension Optional
  • unwrap(_:file:line:)

    Unwraps an optional value. Terminates the process if the value is nil.

    Warning

    While useful for eliminating the need for if let check for guaranteed values, unwrapping nil optionals will result in a crash.

    Declaration

    Swift

    func unwrap(_ because: (() -> String)? = nil, file: StaticString = #file, line: UInt = #line) -> Wrapped

    Parameters

    The reason / expectation for why the value will never be nil.
    file

    The filename where the method is called from.
    line
