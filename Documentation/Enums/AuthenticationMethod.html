AuthenticationMethod

public enum AuthenticationMethod : Sendable

List of authentication methods a client can choose from for Atom configuration.

  • basic(_:)

    Atom will apply basic authorization header to a request on behalf of the client.

    To apply authorization header, Atom needs access to user’s credentials required as BasicCredential (associated value).

    Declaration

    Swift

    case basic(BasicCredential)
  • bearer(_:_:_:)

    Atom will manage access token expiration and apply authorization header to a request on behalf of the client.

    To apply authorization header, Atom needs access to client’s credentials required as ClientCredential (associated value) and the location where to save / read from / updated credentials required as TokenCredentialWritable (associated value). In addition, a valid AuthorizationEndpoint - the location of the authorization server must be provided (associated value).

    Declaration

    Swift

    case bearer(AuthorizationEndpoint, ClientCredential, any TokenCredentialWritable)
  • none

    Application will manage its own authorization headers.

    Declaration

    Swift

    case none
